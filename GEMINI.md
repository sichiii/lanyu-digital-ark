# 海浪上的記憶——達悟族傳統文化數位方舟

## 專案簡介
本專案是一個數位典藏報告網頁，旨在記錄與展示蘭嶼達悟族的核心文化實踐（如拼板舟、飛魚季、地下屋等），並依據傳承危機的緊迫程度進行分類與典藏。

## 技術架構
- **HTML5/CSS3**: 採用 Vanilla CSS 進行視覺設計，利用 CSS 變數管理色彩系統。
- **JavaScript**: 原生 JS 驅動的互動效果與 Canvas 動畫。
- **Metadata 標準**: 遵循 Dublin Core (DC) Metadata 格式。
- **TEK 關聯**: 結合傳統生態知識 (Traditional Knowledge) 標記。

## 檔案說明
- `index.html`: 專案入口整合檔案。
- `index.css`: 全域樣式表。
- `index.js`: 互動邏輯腳本。
- `區塊/`: 其他拆分出的 HTML 組件。
