# 網頁區塊拆解筆記

此專案將原始的 `123.html` 拆解為多個獨立模組，以便於管理與重複使用。

## 檔案結構

- **index.html**: 整合所有區塊的核心檔案。
- **index.css**: 集中管理網頁的所有視覺樣式。
- **index.js**: 包含網頁的所有互動邏輯（如海浪動畫、卡片展開、滾動監測）。
- **頂部提示.html**: 網頁最上方的版權與教育用途聲明。
- **導覽列.html**: 包含導覽列連結。
- **英雄區塊.html**: 網頁首頁的主視覺區塊。
- **故事背景.html**: 典藏緣起與計畫背景介紹。
- **展品選件.html**: 五件核心文化典藏的卡片展示區。
- **傳承危機.html**: 達悟族知識斷層的危機量表圖表。
- **頁腳.html**: 包含授權說明、引用格式與版權資訊。

## 使用說明

1. 若要修改網頁視覺，請編輯 `index.css`。
2. 若要修改功能邏輯，請編輯 `index.js`。
3. 若要更新特定區塊內容（例如增加展品），請直接編輯對應的 `.html` 檔案，並在 `index.html` 中進行整合。
